# Blinkit--Analysis-Python
🛒 Blinkit Data Analysis using Python

📌 Project Overview

This project focuses on analyzing Blinkit grocery sales data using Python. The goal is to extract meaningful insights related to sales performance, product categories, and customer preferences.

---

🎯 Business Objective

To analyze Blinkit's sales data and identify key insights that can help improve business performance, customer satisfaction, and product distribution.

---

📊 Key Performance Indicators (KPIs)

- 💰 Total Sales – Total revenue generated from all items sold
- 📈 Average Sales – Average revenue per transaction
- 📦 Number of Items – Total count of items sold
- ⭐ Average Rating – Average customer rating

---

📈 Analysis & Visualizations

The following analyses were performed:

- 🔹 Total Sales by Fat Content (Donut Chart)
- 🔹 Total Sales by Item Type (Bar Chart)
- 🔹 Fat Content by Outlet for Total Sales (Stacked Column Chart)

---

🧰 Tools & Technologies Used

- 🐍 Python
- 📊 Pandas
- 🔢 NumPy
- 📉 Matplotlib
- 🎨 Seaborn
- 📒 Jupyter Notebook

---

📂 Dataset Information

The dataset includes the following features:

- Item Fat Content
- Item Type
- Outlet Type & Size
- Sales Data
- Ratings

---

🚀 Steps Performed

1. Data Loading
2. Data Cleaning (handling missing values & duplicates)
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights Generation

---

📊 Key Insights

- 🥦 Low fat products contribute significantly to overall sales
- 🍎 Fruits and vegetables are among the top-selling categories
- 🏪 Certain outlet types generate higher sales compared to others
- ⭐ Most products have consistent customer ratings
